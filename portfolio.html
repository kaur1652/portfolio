Lone starfish

For my Level 1 midterm project, I developed a single-page website based on a provided prototype design. The task involved recreating the layout as closely as possible, with the freedom to choose my own subject, images, and colors. I used HTML and CSS to build the structure, organizing files into proper folders and including clear comments in my code. The site features a header with a title overlaid on an image, a text-wrapped article image, a well-structured facts section, and a basic form with correctly linked labels. As this was my first project, I began exploring responsive design and accessibility, gaining valuable hands-on experience with foundational web development concepts.

Find Your Dream Home

For my Level 2 midterm project, I built a responsive 3-page website based on a high-fidelity mockup provided through Figma. This assignment focused on turning a detailed design into a fully functional website using HTML, CSS, and optimized assets. I structured my layout using a combination of CSS Grid and Flexbox to ensure flexibility across different screen sizes. To enhance accessibility, I followed best practices including proper contrast ratios, semantic HTML, alt text for images, skip links, and ARIA roles. I also implemented responsive image techniques, using at least two image sizes to ensure fast loading on both small and large screens. Animations were added to improve the user experience, including one using the animation property. All code was commented to explain my choices, and the final project was deployed using GitHub Pages. This project helped me strengthen my front-end development skills and attention to detail when working from a professional design.

Skills: HTML, CSS
